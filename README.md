[Click here](https://ducks.herokuapp.com/) to check it out on heroku

Skills learnt/used
-----------------
> * Javascript 
> * Jquery

> * Html5

Get the code
------------

Clone the repository:

```shell 
$ git clone 

```

Start the server:

```shell 
$ cd ducks
$ rackup
```

Point your browser to http://localhost:4567/ to play!.
